# 📁 1. STRUKTURA DANYCH (kluczowe)

Zamiast „jakiegoś pliku”, zrób jeden plik:

```text
game_state.json
```

Przykład:

```json
{
  "code": "12345",
  "players": ["Jan", "Ola", "Kuba", "Zosia"],
  "impostors": [1],
  "word": "Samolot",
  "category": "Transport",
  "settings": {
    "num_impostors": 1,
    "hints": true
  },
  "phase": "lobby"
}
```

👉 Ten plik trzymasz na GitHubie i aktualizujesz.

---

# 🧠 2. EKRANY = STANY APLIKACJI

W Streamlicie robisz to przez:

```python
st.session_state["screen"]
```

### Twoje ekrany:

1. `"join"` – wpisanie kodu
2. `"players"` – dodawanie graczy
3. `"settings"` – ustawienia
4. `"game"` – gra
5. `"results"` – wyniki

---

# 🧩 3. EKRAN 1 – KOD GRY

```python
code = st.text_input("Wpisz kod gry (5 cyfr)")

if st.button("Dołącz"):
    if len(code) == 5 and code.isdigit():
        # sprawdź czy istnieje na GitHubie
        # jeśli nie → stwórz nowy JSON
        st.session_state["code"] = code
        st.session_state["screen"] = "players"
```
